# HideTaskBar

Windows 11 でタスクバーを完全に非表示化し、`Win`キーでのみ表示させるツール。

## 機能

- **タスクバー完全非表示** - マウスホバーでも表示されない
- **Winキー押下で表示** - スタートメニューと共にタスクバーが表示
- **自動再非表示** - スタートメニューを閉じると自動的にタスクバーも非表示
- **有効/無効切り替え** - トレイメニューから一時的に無効化可能
- **スタートアップ登録** - Windows起動時に自動実行
- **マルチモニター対応** - 全モニターのタスクバーを制御

## 使用方法

```powershell
cd src/HideTaskBar
dotnet run
```

終了: タスクトレイアイコン → 右クリック → 「終了」

## 技術スタック

- .NET 8 / C#
- WinForms (タスクトレイ用)
- Win32 API (`SetWindowsHookEx`, `ShowWindow`, `SetWinEventHook`)
